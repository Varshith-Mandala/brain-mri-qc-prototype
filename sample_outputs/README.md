## 📂 Sample Outputs

This folder contains example results from the prototype:

- **Good Quality Slice**  
  A high-quality MRI slice with a strong QC score.

- **Bad Quality Slice**  
  A low-quality slice flagged by the QC scoring function.

- **Model Prediction**  
  Example classification output (tumor vs non-tumor) for a given slice.

These outputs demonstrate:
- Basic MRI quality assessment  
- Automated filtering of poor-quality data  
- Feasibility of combining QC with deep learning
